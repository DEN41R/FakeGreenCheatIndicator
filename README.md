 БЫСТРЫЙ ЗАПУСК:

1. Откройте командную строку в папке проекта (src)
2. Выполните команду: dotnet run
3. запустите: bin\Debug\net8.0-windows\FakeCheatIndicator.exe

═══════════════════════════════════════════════════════════

СБОРКА ПРОЕКТА:

Сборка:
  dotnet build

Очистка:
  dotnet clean

Релиз версия:
  dotnet build -c Release

 ИСПОЛЬЗОВАНИЕ:

1. АКТИВАЦИЯ
   - Включите/выключите индикатор одной кнопкой
   - Статус отображается в реальном времени

2. НАСТРОЙКИ
   - Настройте внешний вид и поведение
   - Все изменения применяются мгновенно
   - Не забудьте сохранить настройки кнопкой

3. О ПРОГРАММЕ
   - Информация о версии и возможностях
   - Системная информация

УПРАВЛЕНИЕ ИНДИКАТОРОМ:
- Перетаскивайте индикатор мышью для изменения позиции
- Все настройки применяются в реальном времени
- Индикатор всегда отображается поверх других окон

 ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ:

АРХИТЕКТУРА:
- WPF (Windows Presentation Foundation)
- Material Design In XAML - современная библиотека компонентов
- MVVM-подобная структура - разделение логики и представления
- JSON настройки - сохранение в settings.json
- Win32 API - для создания прозрачного окна индикатора

СТРУКТУРА ПРОЕКТА:
├── App.xaml / App.xaml.cs          # Точка входа приложения
├── MainWindow.xaml / .cs           # Главное окно с навигацией
├── IndicatorWindow.cs              # Окно индикатора
├── Settings.cs                     # Класс настроек
└── FakeCheatIndicator.csproj       # Конфигурация проекта
